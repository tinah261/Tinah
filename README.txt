Projet : Gestion de bibliothèque

Objectif : Le projet consiste à créer un système de gestion de bibliothèque avec des fonctionnalités CRUD (Créer, Lire, Mettre à jour, Supprimer) pour gérer les livres.

Fonctionnalités CRUD de base :
Ajouter un livre : Permet d'ajouter un livre avec des informations comme le titre et l'auteur.
Afficher les livres : Affiche la liste des livres disponibles dans la bibliothèque.
Modifier les informations des livres : Permet de modifier les informations d'un livre existant.
Supprimer un livre : Permet de supprimer un livre de la base de données.
Rechercher un livre : Recherche un livre par son titre ou son auteur.
Technologies utilisées :

Frontend : HTML, CSS, Angular
Backend : PHP
Échange de données : JSON via Fetch API
Fonctions supplémentaires :
Le projet inclut également un système de connexion avec un fichier index.html. Selon le rôle de l'utilisateur, deux interfaces différentes s'affichent :

I) Pour le bibliothécaire :(identifiant = admin et mot de passe = admin)
Les fonctionnalités disponibles sont les suivantes :

Ajouter un livre avec des informations comme le titre et l'auteur.
Afficher la liste des livres.
Modifier les informations des livres.
Supprimer un livre.
Rechercher un livre par titre ou auteur.
Suivi des emprunts : Permet au bibliothécaire de marquer les livres retournés par les utilisateurs.


II) Pour l'utilisateur :(identifiant = user et mot de passe = 1234)
Les fonctionnalités proposées sont les suivantes :

Lister tous les livres disponibles.
Emprunter un livre : Un bouton pour emprunter un livre.
